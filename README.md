# expense-tracker
Personal Expense Tracker is a Python based command-line tool that helps record and manage daily expenses. It uses core programming concepts such as lists,  dictionaries, functions, loops, file handling, and CSV operations

##Key Features 
###1. Add Expense 
Allows the user to enter the date, category, price, and description of each expense. 
###2. View Expenses 
Displays all recorded expenses in an organized format and warns if an entry is incomplete. 
###3. Track Monthly Budget 
Users can set a monthly budget, and the program calculates total expenses and shows remaining or 
exceeded budget. 
###4. Save to CSV 
All expenses can be saved to expense_output.csv using csv.DictWriter, with automatic header creation. 
###5. Load from CSV 
Users can load previously saved expenses back into the program using csv.DictReader. 
###6. Menu-Driven Program 
A simple loop-based menu helps users navigate actions like adding, viewing, saving, tracking, and loading 
expenses.
